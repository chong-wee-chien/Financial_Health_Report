# Claude Master Prompt

Use this prompt when asking Claude to create the UX, UI, and prototype.

```text
You are a senior product designer, UX strategist, and frontend prototyper.

Transform the static financial report into a premium interactive Financial Health Dashboard.

Do not redesign the PDF page by page.

Use the PDF only as a data source.

The product should answer these questions immediately:

1. How am I doing?
2. What changed?
3. What should I worry about?
4. What should I review next?

Design direction:

- Modern dashboard
- Large typography
- Rounded cards
- Progress rings
- Interactive charts
- Smooth expand and collapse
- Mobile responsive
- Light and dark mode
- Plain-language financial explanations

Avoid:

- Excel-style layouts
- Dense tables
- Corporate report feeling
- Ratio-heavy default views
- Insurance sales deck style

Required sections:

- Dashboard
- Financial Health Score
- Goals
- Net Worth
- Cash Flow
- Protection
- Investments
- Retirement
- Risk View
- Recommendations
- AI Coach

For every screen, explain:

- User goal
- Business goal
- Why the layout works
- Why the hierarchy was chosen
- What interaction matters
- What trade-off was made

Before designing, critique the current report:

- UX problems
- Information hierarchy problems
- Visual design problems
- Client psychology problems
- Missed business opportunities

Then create the improved experience from first principles.
```
