# Financial Health Dashboard

This repository is the source of truth for turning a static financial report into an interactive client dashboard.

## Core questions

1. How am I doing?
2. What should I worry about?
3. What should I do next?

## Direction

Move away from static PDF pages, spreadsheet tables, jargon-heavy ratios, and one-time annual reviews.

Move toward a modern dashboard with a health score, plain-language insights, scenario simulation, and clear next steps.
