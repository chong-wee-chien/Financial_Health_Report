# Feature Roadmap

## V1: Demo dashboard

Goal: prove the new experience is stronger than the PDF.

Must include:

- Dashboard
- Financial Health Score
- Goal cards
- Net worth visualisation
- Cash flow visualisation
- Protection cards
- Basic recommendation roadmap
- Responsive design
- Light and dark mode

## V2: Review experience

Goal: make this useful for adviser-client meetings.

Add:

- What changed since last review
- Review meeting mode
- Client notes
- Adviser notes
- Expandable details
- Exportable summary
- Simple sharing flow

## V3: Simulation

Goal: make the product interactive and memorable.

Add:

- Retirement age slider
- Savings adjustment
- Investment return assumption
- Inflation assumption
- Income change scenario
- Major life event scenario

## V4: AI Coach

Goal: allow the client to ask questions in plain language.

Add:

- Floating assistant
- Suggested prompts
- Context-aware answers
- Explanation of sections
- Follow-up questions

## V5: Family and adviser portal

Goal: make the product useful beyond one person.

Add:

- Family dashboard
- Spouse and children profiles
- Household goals
- Adviser task list
- Client follow-up tracking
- Annual review history

## Priority rule

Do not build advanced features before the core dashboard feels clear and premium.

A weak dashboard with many features is worse than a strong dashboard with fewer features.
