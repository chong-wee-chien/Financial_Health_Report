# AI Coach

## Purpose

The AI Coach helps the client understand the dashboard in plain language.

It should explain, guide, and prepare the client for a better review conversation.

## Positioning

The AI Coach is not a replacement for the adviser.

It is a guided explanation layer that helps the client understand their own numbers before and after the review.

## Entry points

- Floating chat button
- Section-level Ask button
- Suggested questions
- Insight cards
- Recommendation cards

## Suggested prompts

- Why is my score this level?
- What changed since last review?
- What should I review first?
- Explain my cash flow simply.
- Explain my CPF simply.
- What happens if I stop working for 6 months?
- Can I retire earlier?
- What does this protection gap mean?

## Answer style

The answer should be:

- Short
- Plain language
- Specific to the user data
- Action-oriented
- Not overly technical

## Guardrails

The AI should not create pressure or make unsupported promises.

It should explain trade-offs and encourage review with the adviser for important decisions.

## UX behaviour

Each section should have context-aware prompts.

Example for Cash Flow:

- Why is my savings rate good or bad?
- Which expense category should I watch?
- How much can I save per month?

Example for Retirement:

- What happens if I retire at 55?
- What if investment return is lower?
- What monthly income can I expect?
