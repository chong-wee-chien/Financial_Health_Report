# Design System

## Direction

The visual style should feel premium, calm, modern, and trustworthy.

Think more like a modern health or finance app, less like a corporate report.

## Layout

Use:

- Large hero section
- Rounded cards
- Clear spacing
- Short sections
- Expandable detail panels
- Sticky navigation where useful

Avoid:

- Dense tables
- Small text
- Overloaded pages
- Screenshot-like PDF layouts

## Typography

Use large typography for key numbers.

Suggested hierarchy:

- Hero number: very large
- Section title: clear and short
- Card title: direct
- Explanation: plain and readable
- Technical detail: hidden behind expand

## Components

Core components:

- Health score card
- Progress ring
- Goal card
- Net worth chart
- Cash flow chart
- Protection card
- Risk scenario card
- Recommendation card
- AI coach button
- Section insight card

## Interaction

Use:

- Hover states
- Smooth expand and collapse
- Animated progress bars
- Chart transitions
- Mobile-friendly cards

## Colour logic

Use colours for meaning, not decoration.

Suggested status mapping:

- Green: strong
- Yellow: watch
- Red: needs review
- Blue: neutral information
- Grey: secondary detail

## Dark mode

Dark mode should be supported from the start.

Cards should remain readable and charts should keep strong contrast.

## Mobile rule

If a section cannot work well on mobile, the design is not ready.
